# ivuBasic

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn dev
```

### Compiles and minifies for production
```
yarn dist
```

### 项目依赖
#### 必要依赖 tm-iview、lodash
```
yarn add tm-iview
yarn add lodash
```
#### 按需依赖 node-emoji
```
yarn add node-emoji
```
